---
title: treeship crew
description: Manage the humans and agents aboard your Treeship.
---

## treeship crew add

Bring a new crew member aboard.

```bash
treeship crew add agent://researcher
treeship crew add human://sarah.chen
```

## treeship crew list

Show all crew members.

```bash
treeship crew list
```

```
ACTOR                    ADDED
human://rezo             2026-03-20T10:00:00Z (captain)
agent://researcher       2026-03-20T10:01:00Z
agent://deployer         2026-03-20T10:01:00Z
human://sarah.chen       2026-03-21T09:00:00Z
```

## treeship crew show

Show details for one crew member.

```bash
treeship crew show agent://researcher
```

```
ACTOR:   agent://researcher
ADDED:   2026-03-20T10:01:00Z
ENTRIES: 47
LAST:    2026-03-26T20:15:00Z
```

## treeship crew remove

Remove a crew member. Existing artifacts remain valid.

```bash
treeship crew remove agent://deprecated-bot
```
