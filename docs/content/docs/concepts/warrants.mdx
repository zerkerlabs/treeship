A warrant is the answer to "who authorized this?" -- signed, scoped, single-use, and verifiable.

## How warrants work

```bash
# Captain issues a warrant
treeship attest warrant \
  --captain human://rezo \
  --description "approve stripe charge max $500 to acme-corp" \
  --expires 2026-03-26T18:00:00Z

# Returns:
# ✓ warrant issued
#   id:    art_warrant_abc123
#   nonce: nce_7f8e9d0a1b2c3d4e
```

The nonce is a one-time token. Pass it to your agent.

```bash
# Agent acts under the warrant
treeship attest action \
  --actor agent://payments \
  --action stripe.charge.create \
  --warrant-nonce nce_7f8e9d0a1b2c3d4e \
  --meta '{"amount":450,"vendor":"acme-corp"}'
```

## What makes it binding

At verification time, the verifier enforces:

```
action.approvalNonce == warrant.nonce
```

<Callout type="warn" title="Nonce binding is mandatory">
This check is in the Rust core and cannot be skipped. If the nonces do not match, verification fails. If the warrant has expired, verification fails. If the same nonce is used twice, verification fails -- the second action has no matching warrant to link to.
</Callout>

## Warrant fields

| Field | Required | Description |
|-------|----------|-------------|
| `captain` | Yes | Human identity URI, e.g. `human://rezo` |
| `description` | Yes | Plain text scope of what is authorized |
| `expires_at` | No | RFC 3339 expiry time |
| `scope.max_amount` | No | Numeric upper bound for financial actions |
| `scope.allowed_actions` | No | Allowlist of action strings |
| `scope.max_actions` | No | Maximum number of actions this warrant covers |

## Scope example

```bash
treeship attest warrant \
  --captain human://sarah.chen \
  --description "approve model inference batch" \
  --expires 2026-03-26T23:59:00Z \
  --scope '{"allowed_actions":["openai.inference","anthropic.inference"],"max_actions":100}'
```

## Verifying a warranted action

```bash
treeship verify art_charge --format json | jq '{outcome, captain: .approver, warrant: .warrant_description}'
# {
#   "outcome": "pass",
#   "captain": "human://rezo",
#   "warrant": "approve stripe charge max $500 to acme-corp"
# }
```
